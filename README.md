# NixOS Hyprland Rice

A self-contained **NixOS + Home Manager** flake for a [Hyprland](https://hypr.land)
desktop, built around a **template-driven theme system**: a single command re-tints
the entire desktop — bar, terminal, notifications, launcher, lock screen, and more —
from one `.theme` palette file.

## What's inside

- **Compositor:** Hyprland, configured in Lua (`home/default/dotfiles/hypr/`).
- **Bar:** Waybar.
- **Terminal:** Kitty.
- **Notifications:** SwayNC (+ Mako template).
- **Launcher / menus:** Rofi (app launcher, theme picker, wallpaper picker, clipboard).
- **Widgets:** AGS / Astal.
- **Lock screen:** Hyprlock (wallpaper-aware, themed).
- **Fetch:** Fastfetch.
- **Shell:** Zsh (+ Starship, zoxide, fzf, eza, etc.).
- **Editor:** Neovim.

### Theme system

`home/default/dotfiles/themes/` holds the heart of the rice:

- `themes/*.theme` — color palettes (ships with `dark-nature`, `rose-pine-moon`,
  `stargazer`, `sunset-room`).
- `templates/*.tpl` — per-app templates with `{{PLACEHOLDER}}` color tokens.
- `apply.sh` — renders every template against the chosen `.theme` and reloads the
  running apps.

Switch themes at runtime:

```sh
~/.config/themes/apply.sh <theme-name>      # e.g. rose-pine-moon
```

…or use the Rofi theme/wallpaper pickers bound in `hypr/keybinds.lua`.

## Layout

```
flake.nix                       # entry point — nixosConfigurations.default
hosts/default/
  configuration.nix             # system config (single user: jaghack)
  hardware-configuration.nix    # PLACEHOLDER — regenerate per machine (see below)
  bootloader.nix                # GRUB (regenerated by install.sh per firmware)
  pkgs/                         # local package derivations
home/default/
  home.nix                      # Home Manager entry
  rice.nix                      # imports all rice modules
  modules/*.nix                 # per-app Home Manager modules
  dotfiles/                     # the actual configs (hypr, waybar, kitty, themes, …)
install.sh                      # interactive single-user installer (live ISO)
```

## Usage

### On an existing NixOS machine

> ⚠️ **`hosts/default/hardware-configuration.nix` is a placeholder stub.** You must
> regenerate it for your hardware before the system will boot:
>
> ```sh
> sudo nixos-generate-config --show-hardware-config \
>   > hosts/default/hardware-configuration.nix
> ```

Then rebuild:

```sh
sudo nixos-rebuild switch --flake .#default
```

The default user is `jaghack` with `initialPassword = "changeme"` — change it after
first boot with `passwd` (or switch to `hashedPassword` / `hashedPasswordFile`).

### Fresh install from a live ISO

`install.sh` partitions a disk, generates the hardware config, regenerates the
bootloader for your firmware (UEFI/BIOS), patches out the NVIDIA block on non-NVIDIA
machines, and runs `nixos-install`. Point it at your own fork:

```sh
FLAKE_REPO=https://github.com/<you>/nixos-rice sudo -E bash install.sh
```

## Notes

- Built against `nixos-unstable`. Adjust monitors in
  `home/default/dotfiles/hypr/monitors.lua` for your displays.
- The optional Fritz!Box reconnect script reads credentials from
  `~/.config/hypr/scripts/.env` (`FRITZ_USER` / `FRITZ_PASSWORD`) — never hardcoded.
- NVIDIA, Tailscale, and a `Europe/Berlin` timezone are enabled by default; tweak in
  `hosts/default/configuration.nix` to taste.
